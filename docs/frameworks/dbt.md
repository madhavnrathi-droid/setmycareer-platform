# DBT — Dialectical Behavior Therapy (RAG knowledge)
Primary source: Linehan (1993, 2014 skills manual — copyrighted; summary only).
Four skill modules (for retrieval):
- **Mindfulness**: observe / describe / participate, non-judgmentally, one-mindfully.
- **Distress tolerance**: TIPP (temperature, intense exercise, paced breathing, paired muscle relaxation); ACCEPTS distraction; radical acceptance for what can't change now.
- **Emotion regulation**: name the emotion → check the facts → opposite action when the emotion doesn't fit facts; PLEASE skills (treat illness, balanced eating, avoid substances, sleep, exercise).
- **Interpersonal effectiveness**: DEAR MAN (describe, express, assert, reinforce, mindful, appear confident, negotiate).
When to surface: emotional flooding, crisis urges, interpersonal conflict, urge-driven behavior.
