# MI — Motivational Interviewing (RAG knowledge)
Primary source: Miller & Rollnick (2013 — copyrighted; summary only).
Spirit: partnership, acceptance, compassion, evocation — never argue for change.
Core skills (OARS): Open questions · Affirmations · Reflections (simple + complex) · Summaries.
Processes: engaging → focusing → evoking (elicit change talk: desire, ability, reasons, need) → planning.
Key moves: roll with resistance; develop discrepancy between values and behavior; scale questions ("how important, 0-10? why not lower?").
When to surface: ambivalence about change (habits, substances, avoidance), low motivation, pre-contemplation.
