# CBT — Cognitive Behavioral Therapy (RAG knowledge)
Primary sources: Beck (1979); CCI workbooks cached in `data/raw/frameworks/`.
Core loop: situation → automatic thought → emotion → behavior. Change the thought or the behavior, the emotion follows.
Key techniques (for retrieval):
- **Cognitive restructuring**: identify automatic thought → find the distortion (catastrophizing, all-or-nothing, mind-reading, fortune-telling, personalization, should-statements) → examine evidence for/against → balanced alternative thought.
- **Thought record**: situation / thought / emotion (0-100) / evidence / alternative / re-rate.
- **Behavioral activation**: schedule small values-aligned activities BEFORE motivation returns; action precedes mood (Jacobson 1996; Cuijpers 2007 meta-analysis).
- **Behavioral experiments**: test the prediction ("if I speak up, everyone will laugh") with a small real-world trial.
When to surface: rumination, distorted self-talk, avoidance, low-mood inactivity.
