# Evidence-based guideline anchors (RAG knowledge)
- **WHO mhGAP v2** (cached PDF, CC BY-NC-SA 3.0 IGO): assessment + management protocols for depression, self-harm/suicide, stress; designed for non-specialist settings — good conservative grounding.
- **NICE guidance** (access via nice.org.uk — UK terms; no bulk redistribution): NG222 (depression in adults) and CG113 (GAD/panic) recommend stepped care: low-intensity first (guided self-help, BA, CCBT), escalate to high-intensity CBT/medication review. Cite, link, don't republish.
- Safety rule for all nodes: any suicidality/self-harm signal → route to human crisis resources (988 in US), never to a coping-tip response alone.
